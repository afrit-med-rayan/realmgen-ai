# RealmGen AI

Procedural Fantasy World Generator built with Python and PySide6.

## Features
- **Deterministic Generation**: Seed-based terrain and population.
- **Dynamic Lore**: Context-aware history for every location.
- **Interactive Map**: Zoom, pan, and click to view details.
- **Filtering & Search**: Easily navigate and find locations.
- **Save & Export**: Save worlds to JSON or export map as PNG.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
